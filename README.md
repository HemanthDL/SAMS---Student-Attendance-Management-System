# SAMS---Student-Attendance-Management-System

Developed an Application which maintains the attendance data of <br>
Students . The primary aim of this project is to create an integrated <br>
attendance management system that simplifies and automates the <br>
process of recording and calculating student attendance. This <br>
application allows the teacher to mark attendance and view <br>
attendance report of all student . <br>
Student have the access to view his/her attendance report for all <br>
possible subject. <br>
Technologies used : HTML , CSS , JavaScript , MYSQL with PHP <br>
